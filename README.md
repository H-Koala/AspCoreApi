#### 项目介绍
* Restful API
* AufoFac支持
* 封装好Dapper，开箱即用(Oracle+MMSYQL+MYSQL支持)
* Dapper事务管理，仅需给方法打特性标签[WorkCommit]
* 使用了MiniProfile 对Dapper or EF Core 生成的SQL 进行监控
* NLog支持
* JWT支持
* RabbitMQ支持
* Swagger 可视化API文档
* AOP,拦截器，自定义中间件等支持
